# 多语言单词文章生成AI应用 - 重新设计说明

## 核心功能

1. **文章生成AI**：根据用户提供的单词、水平、偏好生成文章
2. **多语言支持**：支持AI可以支持的所有语言（基于提示词模板）
3. **移动端支持**：响应式设计，支持PWA离线使用
4. **离线/在线模式**：支持本地AI模型和在线API

## 技术架构

### 后端
- Python Flask API
- 支持多种AI模型（Ollama本地、OpenAI、Claude等）
- 多语言提示词模板系统
- 文章生成引擎

### 前端
- React + TypeScript
- 响应式设计（移动端优先）
- PWA支持（离线功能）
- 多语言界面

## 提示词模板系统

基于用户提供的孟加拉语提示词模板，为每种语言创建对应的提示词模板。
